# Xin chào, mình là Chiến (Trần Trọng Chiến)
### Sinh viên CNTT tại Đại học Mở Hà Nội (FITHOU)
Mình là một nhà phát triển Full-stack với niềm đam mê xây dựng các giải pháp
đô thị thông minh và hệ thống quản lý hiệu quả.
- Mình hiện đang tập trung vào: **Smart City Infrastructure & Linked Open
Data**.
- Thành thạo các tiêu chuẩn: **NGSI-LD, SOSA/SSN**.
- Tech stack yêu thích: **PHP (CodeIgniter), Python (FastAPI), Java
(Android)**.
- Luôn sẵn lòng thảo luận về: **Agile/Scrum, DevOps & CI/CD**.
